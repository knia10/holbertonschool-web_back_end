## Learning Objectives
- 🔰 How to run a Redis server on your machine

- 🔰 How to build a basic Express app interacting with a Redis server

- 🚩 All of your code will be compiled/interpreted on Ubuntu 18.04, Node 12.x, and Redis 5.0.7
